IDE's, GitHub e mais

Aqui você encontra uma seção bolachinha de dicas de IDE's, GitHub e benefícios educacionais

JetBrains Student

Com o seu e-mail educaional (cpf@veigadealmeida.edu.br), você tem direito a um ano (e podendo renovar até o final do curso) as IDE's pagas da JetBrains

GitHub Educational

Usando o seu e-mail educacional, você também consegue benefícios do GitHub Students, como cursos exclusivos, certficados e vouchers de exames de certficação. Saiba mais: https://github.com/education

Microsoft 365 (Office)

Você aluno também tem direito, enquanto matrículado no curso, o uso do Microsoft 365, antigo Office.
